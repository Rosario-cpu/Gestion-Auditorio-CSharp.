 Gestión de Auditorio en C#

## Descripción del Proyecto
Este proyecto es una aplicación de consola desarrollada en **C#** que simula un sistema de gestión de asistentes para un auditorio. El objetivo principal es garantizar que el orden de ingreso se respete mediante el uso de una estructura de datos tipo **Queue (Cola)**, aplicando el principio **FIFO** (*First In, First Out* - Primero en entrar, Primero en salir).

## Características Principales
* **Implementación de Estructuras de Datos:** Uso eficiente de la clase `Queue` de `System.Collections.Generic` para el control de flujo de personas.
* **Programación Orientada a Objetos:** Código modular organizado en clases (`Asistente`, `Tools`, `Program`) para una mejor escalabilidad y mantenimiento.
* **Interfaz de Consola:** Sistema interactivo que permite visualizar el estado de la cola en tiempo real.

## Tecnologías Utilizadas
* **Lenguaje:** C# (.NET)
* **Herramientas:** Visual Studio Code, Git, GitHub.
